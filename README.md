# Jennie Wolff — Website

Holistic brand and leadership coaching website for Jennie Wolff / Joyous Wolf Consulting LLC.

## Tech Stack

- React + TypeScript
- Tailwind CSS v4
- Framer Motion
- React Router
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Connected to Vercel via GitHub. Every push to `main` deploys automatically.
